JavaScript: Variables, Conditions, and Loops

This page demonstrates basic JavaScript programming concepts with variables, conditional logic, and loops.

1. Variables

Variables store values so the program can use them later. In JavaScript, use let or const.

const name = 'Aisha';
let age = 18;
let isStudent = true;

2. Conditions

Use if and else to make decisions based on values.

if (age >= 18) {
  console.log('You are an adult');
} else {
  console.log('You are a minor');
}

3. Loops

Loops repeat code until a condition becomes false.

for (let i = 1; i <= 5; i++) {
  console.log(i);
}
