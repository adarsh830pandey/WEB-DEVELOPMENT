🎯 DOM Manipulation in JavaScript

1. Selecting DOM Elements

// By ID document.getElementById('myId') // By Class document.getElementsByClassName('myClass') document.querySelectorAll('.myClass') // By Tag document.getElementsByTagName('div') // Universal selectors document.querySelector('.myClass') // first match document.querySelectorAll('div > p') // all matches

2. Accessing Element Properties

let element = document.getElementById('myElement'); // Content element.textContent // text only element.innerHTML // with HTML tags element.innerText // rendered text // Attributes element.id element.className element.getAttribute('data-value')
Sample Element with Content

3. Modifying Element Content

// Text content element.textContent = 'New text' // HTML content element.innerHTML = '<strong>Bold Text</strong>' // Insert HTML element.insertAdjacentHTML('beforeend', '<p>New</p>')
Original Content

4. Modifying Attributes

// Set attribute element.setAttribute('id', 'newId') element.getAttribute('id') element.removeAttribute('id') // Direct property element.id = 'newId' element.className = 'myClass' // Check attribute element.hasAttribute('data-value')
Demo Element

5. Modifying Styles

// Inline styles element.style.color = 'red' element.style.backgroundColor = '#f0f0f0' element.style.padding = '10px' // Get computed styles getComputedStyle(element).color getComputedStyle(element).backgroundColor
Styled Demo

6. Modifying CSS Classes

// Add class element.classList.add('active') // Remove class element.classList.remove('highlight') // Toggle class element.classList.toggle('hidden') // Check class element.classList.contains('active')

// Create element let div = document.createElement('div') div.textContent = 'New Element' // Insert into DOM parent.appendChild(div) parent.insertBefore(div, sibling) element.insertAdjacentElement('beforeend', div)
