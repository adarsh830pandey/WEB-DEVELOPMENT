⚛️ React Components & JSX Basics

1. Functional Components

// Simple functional component function Welcome() { return <h2>Hello, React!</h2> } // Export component export default Welcome

2. JSX Syntax Basics

// JSX looks like HTML but it's JavaScript function App() { return ( <div> <h1>Welcome to React</h1> <p>JSX is HTML-like syntax in JavaScript</p> </div> ) } // JSX must return single root element

3. Components with Props

// Component receives props function Greeting(props) { return <h2>Hello, {props.name}!</h2> } // Using component <Greeting name="John" /> <Greeting name="Jane" />

4. Props Destructuring

// Destructure props in function parameters function Card({ title, description, color }) { return ( <div style={{ background: color }}> <h3>{title}</h3> <p>{description}</p> </div> ) }

5. Default Props

// Set default values for props function Button({ label = "Click Me", color = "blue" }) { return ( <button style={{ background: color }}> {label} </button> ) } // With defaults <Button /> <Button label="Submit" color="green" />

6. Props Validation (PropTypes)

import PropTypes from 'prop-types' function User({ name, age, email }) { return ( <div> <p>Name: {name}</p> <p>Age: {age}</p> <p>Email: {email}</p> </div> ) } User.propTypes = { name: PropTypes.string.isRequired, age: PropTypes.number, email: PropTypes.string }

7. Children Props

// Access content between component tags function Container({ children, title }) { return ( <div style={{ border: '1px solid #ccc' }}> <h3>{title}</h3> {children} </div> ) } // Usage <Container title="Box"> <p>Content inside</p> </Container>

8. Conditional Rendering

// Show/hide based on condition function StatusMessage({ isOnline }) { return ( <div> {isOnline ? ( <p>✓ User is online</p> ) : ( <p>✗ User is offline</p> )} </div> ) } // Or using && operator {isOnline && <p>Welcome back!</p>}

9. Rendering Lists

// Render list of items function UserList({ users }) { return ( <ul> {users.map(user => ( <li key={user.id}>{user.name}</li> ))} </ul> ) } // Usage const users = [ { id: 1, name: 'John' }, { id: 2, name: 'Jane' } ] <UserList users={users} />

10. Styling Components

// Inline styles (object) function StyledBox() { const styles = { container: { background: '#667eea', padding: '20px', color: 'white', borderRadius: '8px' } } return <div style={styles.container}>Styled Box</div> } // CSS classes <div className="my-class">Content</div>

11. Event Handling in Components

// Handle click events function Counter() { const [count, setCount] = React.useState(0) return ( <div> <p>Count: {count}</p> <button onClick={() => setCount(count + 1)}> Increment </button> </div> ) }

12. Component Composition

// Combine components function Profile({ user }) { return ( <div> <Header name={user.name} /> <Bio description={user.bio} /> <Footer email={user.email} /> </div> ) } // Reusable parts function Header({ name }) { return <h1>{name}</h1> }

13. Higher Order Components (HOC) Pattern

// HOC adds functionality to component function withLogger(Component) { return (props) => { React.useEffect(() => { console.log('Component mounted') return () => console.log('Component unmounted') }, []) return <Component {...props} /> } } // Usage const EnhancedComponent = withLogger(MyComponent)

14. Custom Hooks for Reusable Logic

// Create custom hook for form function useFormInput(initialValue) { const [value, setValue] = React.useState(initialValue) return { value, bind: { value, onChange: e => setValue(e.target.value) } } } // Use hook function Form() { const name = useFormInput('') return <input {...name.bind} /> }

15. React Fragment (& shorthand)

// Return multiple elements without wrapper function List() { return ( <> <h2>List</h2> <ul>...</ul> </> ) } // Or explicit Fragment <React.Fragment> <h2>Title</h2> <p>Content</p> </React.Fragment>
