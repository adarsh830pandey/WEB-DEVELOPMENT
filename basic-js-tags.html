Basic JavaScript Tags

Learn how JavaScript works inside HTML using <script> tags.

1. What is the <script> tag?

The <script> tag tells the browser to run JavaScript code. It can contain code directly inside the tag or load code from an external file.

2. Inline JavaScript

This is the most basic way to add JavaScript to a page:

<script>
  alert('Hello from inline JavaScript!');
</script>

The code above runs immediately when the browser reaches the script.

3. External JavaScript file

Use the src attribute to load JavaScript from a separate file:

<script src="js-basics.js"></script>

External scripts keep HTML clean and help reuse code across pages.

4. Adding scripts at the end of <body>

Put your script before the closing </body> tag so the page content loads first. That way, JavaScript can safely access page elements.

This example uses an inline <script> that attaches a listener to the button.

<script>
  const button = document.getElementById('demoButton');
  const output = document.getElementById('demoOutput');

  button.addEventListener('click', () => {
    output.textContent = 'The JavaScript button was clicked!';
  });
</script>

6. Using defer and async

For external scripts, you can control loading timing with optional attributes:

  • defer: download the script while the page loads, then run it after parsing HTML.
  • async: download the script while the page loads and run it as soon as it finishes downloading.
<script src="js-basics.js" defer></script>
<script src="js-basics.js" async></script>

7. Example of an external script

This page also loads an external script file from js-basics.js.

// js-basics.js
document.addEventListener('DOMContentLoaded', () => {
  const message = document.getElementById('externalMessage');
  if (message) {
    message.textContent = 'This message was set by an external JavaScript file.';
  }
});

8. Why use JavaScript tags?

JavaScript tags make it possible to:

  • Show messages
  • React to user clicks
  • Change content dynamically
  • Validate forms
